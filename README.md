# Kalah Game

Web application that implements the Kalah Game: https://en.wikipedia.org/wiki/Kalah

## Run

mvn clean install

java -jar target/kalah-1.0-SNAPSHOT.jar 

The game can then be accessed at: http://localhost:8080
Each browser tab will host a game.


## Stack
* Java
* SpringBoot
* SockJS
* HTML5
* CSS
* JavaScript
* JQuery
